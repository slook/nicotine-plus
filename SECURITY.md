# Security Policy

## Supported Versions

Nicotine+ follows a `A.B.C` versioning scheme. We only support the latest
released `A.B.x` series, which is currently `3.3.x`.

## Reporting a Vulnerability

Please report a vulnerability under the Security tab in the GitHub repository.
Since Nicotine+ is a volunteer-driven project, your report will be reviewed and
addressed when the maintainers have time.
